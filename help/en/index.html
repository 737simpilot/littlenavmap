Little Navmap Help

Version 0.9.5.develop

Installation

An installer or setup program is not available.

Do not extract the archive into the folder c:\Program Files\ or c:\Program Files (x86)\ since you will need administrative privileges in some Windows versions. Since Windows keeps control on these folders other problems might occur like replaced or deleted files.

Extract the Zip archive into a folder like c:\Own Programs\Little Navmap. Then start the program by double-clicking littlenavmap.exe.

Little Navmap is a 32-bit application and was tested with Windows XP 32-bit, Windows 7 64-bit and Windows 8.1 64-bit.

Features

General

  • Supported platforms: Windows XP/7/8/10, Linux, Mac OS X
  • Conservative user interface without experiments: Using a main menu, context menus, toolbars, dock windows, undo, recent file lists and more.
  • Written using C++ and Qt - it is fast
  • The program saves almost all settings, like flight plan, windows positions, searches and more and restores everything on startup
  • Most question or information dialogs can be disabled.
  • Support for Flight Simulator X, Flight Simulator - Steam Edition, Prepar3D v2 and Prepar3D v3 (partially untested)
  • Weather display from ActiveSkyNext, NOAA or Vatsim in tooltips and information panels
  • Databases for simulators are kept separate and can be changed on the fly
  • Little Navmap can be run on computers that do not contain any simulator installation at all. Just copy the database from your flight simulator computer.
  • Airport and navaid icons used throughout the whole program for easier recognition
  • Information panels showing all about airports or navaids
  • Exhaustive search function for airports and navaids that allows to filter by an unimaginable amount of options
  • Distance based search for airports or navaids around a center point. Can be combined with all other search options.

Flight Plan and routing

  • Can read and write FSX PLN format (XML).
  • Drag and drop flight plan editing on the map
  • Easy flight plan editing in the table view, like moving and deleting waypoints
  • Automatic routing (does not require online services) for high/low altitude airways, VOR/NDB and preset cruise altitude
  • Elevation profile display for flightplan
  • Undo/redo for flight plan edits
  • Robust against malformed flight plans - does not put unknown waypoints "into the desert"

Map

  • No need to download and install any additional elevation or map data.
  • Uses two free online maps (OpenStreetMap amd OpenTopoMap) as well as two simple offline maps
  • Multiple map themes including hillshading and two map projections selectable
  • Airport display iconography based on real world VFR maps thus combining plenty of information in a small symbol
  • Detailed airport diagrams displayed in the map (not a separate window) including parking, taxiways, displaced threshold, overrun areas and much more.
  • Add-on airports are highlighted in a different text format for easier regognition
  • History of map position and zoom levels like a web browser.
  • Distance measurement lines and range rings
  • Map display configuration including detail level on the toolbar
  • Symbol and text sizes configurable in options dialog
  • Can load and display Google Earth KML files
  • Detailed tooltips on map display

Simulator aircraft

  • Shows simulator aircraft on the map including flight plan progress display (needs Little Navconnect)
  • Track of simulator aircraft
  • Using Little Navconnect as an agent between the simulator and Little Navmap saves the pain of remote simconnect setup

General Remarks

The user interface of Little Navmap consists of multipe dock windows that are arranged around the main window containing the map view. The dock windows can be detached from the main window and also moved around in their docked position. All docks can be closed if they are not needed. You can even drop docks on each other to create a tabbed view (tabs will appear at the bottom of the dock stack). Toolbars are also movable if you click on the left handle and can be closed or undocked from the main window too.

Please note that due to a variety of external data like KML or BGL files that can be loaded by Little Navmap crashes or malfunctions can occur. If that is the case send me the offending file and Little Navmap's log file and configuration which can be located in the about dialog. My e-mail address is shown in the about dialog of Little Navmap too.

First Start

When starting the first time Little Navmap will show the Load Scenery Library dialog. From there your can select all recognized Flight Simulators and load their scenery libaries into Little Navmap's internal database. One database is kept for each simulator and it can be changed on the fly in the Scenery Library menu.

See the description of the Scenery Library for more information.

See chapter Running without Simulator Installation for information how to run Little Navmap without a Flight Simulator installation.

Quick Overview

Menus and Toolbar

This chapter describes the menu structure of Little Navmap. You will find parts of this functionality on the toolbars as well which are not be described separately. Key combinations can be seen on the menu items and are not listed separately.

File

Flight Plans

You can load, save or create new FSX PLN files. FS9 PLN files are not supported.

An opened flight plan file will be reloaded on startup (reload and centering can be switched off in the Options dialog).

Google Earth KML Files

You can add one or more Google Earth KML or KMZ file to the map display. All added KML or KMZ files will be reloaded on startup (reload and centering can be switched off in the Options dialog).

All loaded KML files can be removed from the map with the menu item Clear KML from Map

Work Offline

Stops loading of map data from the internet. This affects only the OpenStreetMap and OpenTopoMap map themes.

Flight Plan

Undo/Redo

Allows to undo and redo all flight plan changes.

Select a Start Position for Departure

A parking/gate/ramp, runway or helipad can be selected as a start position on the departure airport. A parking position can also be selected in the map context menu item Set as Flight Plan Departure. If no position is selected the longest primary runway end is selected automatically as start.

Edit Flight Plan on Map

Switches flight plan edit mode on the map on or off. See Flight Plan Editing.

Calculate Direct

Deletes any intermediate waypoints and connects departure and destination using a great circle route.

Calculate Radionav

Creates a flight plan that uses only VOR and NDB stations as waypoints and ensures that reception of at least one station is ensured along the whole flight plan. Note that VOR stations are preferred before NDB and DME stations are not used at all. Calculation will fail if not enough radio navaids can be found between departure and destination.

Calculate high Altitude

Uses Jet Airways to create a flight plan. The resulting minimum altitude is set into the flight plan altitude field. A simplified east/west rule is used to adjust the cruise altitude (can be switched off in the Options dialog).

Calculate low Altitude

Uses Victor Airways to create a flight plan. Everything else is the same as in Calculate low Altitude.

Calculate based on given Altitude

Use the value in the altitude field of the flight plan to find a flight plan along Victor and/or Jet airways. If the altitude value is too low calculation will fail.

Reverse Flight Plan

Swaps departure and destination and reverses order of all intermediate waypoints.

Map

Goto Home

Goes to home area Set Home using the saved position and zoom factor.

Go to Center for Distance Search

Go to the center point used for distance searches.Set Center for Distance Search

Center Flight Plan

Shows the whole flight plan on the map.

Center Aircraft

Zooms to the aircraft if connected to a Flight Simulator and keeps the aircraft centered.

Delete Aircraft Trail

Removes the aircraft trail if connected to a Flight Simulator.

Map Position Back Forward

Jumps forward or backward in the map position history. The complete history is saved and restored when starting Little Navmap.

More/Default/Less Details

Increases or decreases details on the map. More details mean more content, more text information and bigger icons.

Projection

Mercator

A flat projection that gives the most fluid movement when using the OpenStreetMap and OpenTopoMap themes.

Spherical

Shows earth as a globe which is the most natural projection. Movement can stutter when using the OpenStreetMap and OpenTopoMap themes. Use the Simple or Plain map themes to prevent this.

Theme

OpenStreeMap

This is an online raster map that includes and hillshading option. Note that hillshading does not cover the whole globe.

OpenTopoMap

A beautiful online raster map that aims to mimic a paper topographic map as much as possible. Includes hillshading and elevation contour lines at lower zoom levels. Disadvantage is that airports and navaids are hard to recognize in the colorful map.

Simple

This is an offline political map using colored countries. The map is simple and included in Little Navmap. Has an option to display city and country names.

Plain

A very simple offline map. The map is and included in Little Navmap. Has an option to display city and country names.

Force Show Addon Airports

If this option is selected add-on airports are always shown independent of the other airport map settings.

Show Airports with hard Runways

Show airports that have at least one runway with a hard surface.

Show Airports with soft Runways

Show airports that only soft surfaced runways.

Show empty Airports

Shows empty airports. This button can be hidden depending on settings in the Options dialog.

An empty airport is defined as one which has neither parking nor taxiways nor aprons and is no add-on. These airports are treated differently in Little Navmap since they are the most boring of all default airports.

Show VOR Stations, Show NDB Stations, Show Waypoints, Show ILS Feathers, Show Victor/Jet Airways, Show Flight Plan, Show Aircraft Trail

Simply shows or hides these facilities or navaids on the map.

Show Map Grid

Shows a latitude/longitude grid as well as the meridian and anti meridian (date line) on the map.

Show Country and City Names, Show Hillshading

Availability of these options depends on the selected map theme. See Theme.

Map Display

Moving

Use click an drag to move the map and the mousewheel to zoom in or out. You can also use the overlay buttons on the right side of the map.

Use the cursor keys to scroll the map and + and - to zoom in and out. Use Alt+Left and Alt+Right to go forward or backward in the map position history. Do not forget to activate the map window by clicking into it before using any keys for movements.

Tooltips

Hovering the mouse over the map will show tooltips for all objects like aiports, VOR, airways and more nearby the cursor. The tooltip is truncated if it gets too long. In that case reduce details or zoom in. The sensitivity for tooltip display can be adjusted in the Options dialog.

Mouse Clicks

A single click on an airport or navaid shows details in the information dock window. A double click zooms in showing either the airport diagram or the navaid closely and additionally shows details in the information dock window.
