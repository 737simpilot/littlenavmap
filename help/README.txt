Place a PDF manual "little-navmap-user-manual-XX.pdf" with language code "XX" in 
this directory to be used on computers that have this language setting. 
A file name "legend-XX.html" has to exist too.

An empty file with name "little-navmap-user-manual-XX.online" indicates 
that the language "XX" is also available online on Gitbook. 
This affects all help buttons and the links in the help menu which use the online help on Gitbook.
